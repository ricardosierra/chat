## Instalando no Windows

### Pré Requisitos para o Windows

- Requer Mongo

### Configurar mongodb service

```bash
mongod --install --logpath c:\data\mongodb.log
```

### Inicializa com os seguintes comandos

```bash
net start mongoDB
nodemon index
```
